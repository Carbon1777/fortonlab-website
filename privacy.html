1. Оператор

Оператор персональных данных: Forton Lab — независимая студия мобильных приложений (Российская Федерация).

Контакт для запросов GDPR / 152-ФЗ: info@fortonlab.ru (с пометкой «Privacy» в теме письма).

Сайт: https://fortonlab.ru

2. Какие данные собирает сайт

Сайт fortonlab.ru является информационной визиткой студии. Мы НЕ собираем регистрационные данные пользователей, НЕ принимаем платежи на сайте, НЕ ведём аналитику с идентификацией пользователей.

Server logs (IP-адрес, User-Agent, время запроса) хранятся на стороне CDN-провайдера Cloudflare не более 30 дней (стандарт Cloudflare Free Tier).

Cookies: только service cookies Cloudflare для DDoS-защиты (например, __cf_bm). Аналитические и рекламные cookies не используются.

3. Использование Google API Services

Сервис автоматизированной публикации Forton Lab Publisher использует YouTube Data API v3 (Google API Services) для загрузки видео-материалов студии на YouTube-канал @fortonlab, владельцем которого является администратор Forton Lab. Использование данных, полученных от Google APIs, регулируется Google API Services User Data Policy, включая требования Limited Use.

Какие данные обрабатываются:

  • Refresh token и access token для авторизации в YouTube Data API
  • Метаданные публикуемых видео (заголовок, описание, теги, категория)

Forton Lab Publisher НЕ:

  • получает доступ к видео других пользователей
  • не собирает персональные данные подписчиков канала
  • не передаёт никакие данные третьим сторонам
  • не использует данные для рекламы, ретаргетинга или персонализации

Хранение токенов: refresh token хранится в зашифрованном виде в GitHub Secrets репозитория студии. Доступ ограничен администратором студии.

Удаление данных: запрос на удаление обрабатываемых через Google API данных можно направить на info@fortonlab.ru. Срок обработки — 14 дней.

4. Безопасность

Информация передаётся только по HTTPS. Токены доступа Google API хранятся в защищённом GitHub Secrets и недоступны третьим лицам. Доступ к репозиторию ограничен администратором студии и защищён двухфакторной аутентификацией.

5. Срок хранения и удаление

Refresh token хранится бессрочно до момента отзыва пользователем (через https://myaccount.google.com/permissions) или администратором студии.

По запросу через info@fortonlab.ru данные могут быть удалены в течение 14 дней.

6. Права субъекта персональных данных (152-ФЗ)

В соответствии с Федеральным законом РФ № 152-ФЗ «О персональных данных» субъект персональных данных имеет право:

  • на доступ к своим персональным данным;
  • на исправление неточных или неполных данных;
  • на удаление персональных данных;
  • на отзыв согласия на обработку персональных данных;
  • на жалобу в Роскомнадзор (надзорный орган в области защиты прав субъектов ПД).

Контакт для реализации прав: info@fortonlab.ru (с пометкой «Privacy» в теме письма).
